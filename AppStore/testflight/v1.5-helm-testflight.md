# TestFlight 1.5 (build 29) — Helm CLI

Last updated: 2026-07-10

**Scope:** Beta only via TestFlight. Uses App Store Connect version **1.5** drafts (iOS + macOS). Do not submit for App Review.

## What to Test

Helm applies beta notes from [`beta-notes/en-US/whats-new.txt`](beta-notes/en-US/whats-new.txt) during upload (`--whats-new-path beta-notes`). **English only** — TestFlight can store per-locale notes, but testers typically see the primary English copy; localized App Store What's New (es/ca) is separate and only matters at App Store release.

### English (en-US)

```text
This update makes RetroRapid! more stable and consistent.

We fixed a crash that could occur when sound effects or haptic cues were playing, and aligned in-app branding so menus and settings show RetroRapid! correctly throughout.

Game Center achievements, friend markers, and sharing are unchanged — still the best way to chase your crew.

What we'd like you to focus on:
- Play several runs with sound and haptic lane cues enabled
- Check Settings and About for consistent RetroRapid! branding
- Report any unexpected crashes, especially around game-over or lane changes

Thanks for racing with us.
```

## Helm workflow

Uses the **Mestre hybrid pattern** (proven from Cursor agent uploads):

1. `xcodebuild archive` with **Xcode 26** (`DEVELOPER_DIR=/Applications/Xcode.app`)
2. `xcodebuild -exportArchive` with `ExportOptions-upload.plist` (`destination: upload`)
3. `helm-asc build update` — export compliance + en-US What to Test
4. `helm-asc build attach` — External Testing group

`helm-asc builds upload` is not used here; Helm’s file sandbox blocks agent/IDE shells from staging `.ipa`/`.pkg` in the CLI Inbox.

```bash
# Full pipeline
swift run --package-path Scripts submit-testflight-build all

# Or step by step (if archives already exist):
swift run --package-path Scripts submit-testflight-build upload-ios
swift run --package-path Scripts submit-testflight-build upload-mac
```

Beta groups (defaults):

| Group | ID |
|---|---|
| Internal Testing | `47135367-a693-4b8f-ba0b-01ed017148a2` |
| External Testing | `df40f833-12c7-4411-b28d-122690045c58` |

Override groups:

```bash
swift run --package-path Scripts submit-testflight-build upload-ios \
  --external-group 47135367-a693-4b8f-ba0b-01ed017148a2
```

Preview without archiving, uploading, or changing App Store Connect:

```bash
swift run --package-path Scripts submit-testflight-build upload-ios --dry-run
```

Manual Helm equivalents:

```bash
HELM=/Applications/Helm.app/Contents/Helpers/helm-asc
APP=6758641625
NOTES=AppStore/testflight/beta-notes
GROUPS=47135367-a693-4b8f-ba0b-01ed017148a2,df40f833-12c7-4411-b28d-122690045c58

$HELM apps $APP builds upload \
  --platform iOS --version 1.5 --number 29 \
  --file build/testflight-1.5/export-ios/*.ipa \
  --wait-for-processing --uses-non-exempt-encryption false \
  --whats-new-path "$NOTES" --groups "$GROUPS" \
  --submit --auto-notify true

$HELM apps $APP builds upload \
  --platform macOS --version 1.5 --number 29 \
  --file build/testflight-1.5/export-mac/*.pkg \
  --wait-for-processing --uses-non-exempt-encryption false \
  --whats-new-path "$NOTES" --groups "$GROUPS" \
  --submit --auto-notify true
```

## Platforms

|---|---|---|---|

| Mac | `macOS` | `generic/platform=macOS` | Separate archive required |

tvOS and visionOS are not part of this TestFlight drop.
